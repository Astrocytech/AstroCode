## DIFF FORMAT RULES
1. Start with `--- a/filepath` and `+++ b/filepath`
2. Use `@@ -old +new @@` hunk header
3. Lines starting with `-` are removed
4. Lines starting with `+` are added
5. Context lines (unchanged) have no prefix
